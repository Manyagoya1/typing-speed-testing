# Typing Speed Test (Python Project)

##  PROJECT OVERVIEW
A simple typing speed tester made using Python.  
The program shows a random sentence, times the user's typing, and calculates **WPM** and **accuracy**.

## FEATURES
- Random sentence generator
- Calculates typing speed (words per minute)
- Calculates accuracy
- Multiple tries (loop-based)
- Easy Python code using if/elif/else and loops
- Beginner-friendly logic

## TECHNOLOGIES USED
- Python 3
- time module
- random module

## HOW TO INSTALL AND RUN ?
1. Download or clone this repository:
   https://github.com/Manyagoya1/typing-speed-testing
2.Open the folder:
   cd typing_speed_test

3. Run the program
- The program displays a sentence.
- Press ENTER to start.
- Type the given sentence.
- It shows your WPM, accuracy, and time.
- Program asks if you want to try again.

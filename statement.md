# Problem Statement
Typing speed is an essential skill for students and professionals.  
Many existing typing tools are complex or require internet access.  
This project aims to create a simple offline typing-speed tester using Python.

# Scope of the Project
- Beginner-friendly
- Works on any computer with Python installed
- Uses basic programming logic (if, elif, else, loops)
- Measures typing speed and accuracy

# Target Users
- Students learning Python
- People who want to improve typing speed

# High-Level Features
- Random sentence selection from a list
- Timer-based speed calculation
- Accuracy checking (character comparison)
- Word-per-minute output
- Loop-based retry system
